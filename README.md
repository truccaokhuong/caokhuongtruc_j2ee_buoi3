# caokhuongtruc_j2ee_buoi3

Spring Boot + Thymeleaf - Demo quản lý sách

## Chức năng

- Hiển thị thông điệp "xin chào các bạn" tại trang chủ
- Demo truyền dữ liệu qua Model (Student)
- CRUD quản lý sách (Book): Thêm, Sửa, Xóa, Danh sách

## Công nghệ

- Spring Boot 3.3.2
- Thymeleaf
- Maven
- Java 17/22

## Chạy ứng dụng

```bash
mvn spring-boot:run
```

Sau đó truy cập:
- Trang chủ: http://localhost:8080
- Demo: http://localhost:8080/demo
- Quản lý sách: http://localhost:8080/books
